## N+1 문제점 예시
처음 프로젝트 설계시 게시판 글 조회 시 댓글 목록도 함께 조회를 하고자 하였습니다. 그러자 한번 게시글 조회시 해당 게시글에 작성된 모든 댓글들도 함께 조회가 되었습니다.
그리고 이 부분들이 한번에 조회된 것이 아닌,
- 게시글 조회 : 게시글에 작성된 댓글의 id가 함께 조회
- 각 댓글 조회 : 게시글 조회시 조회된 댓글 id를 통해 댓글 각각 조회
